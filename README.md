## 启动服务

### pm2启动
1. 安装pm2
~~~shell
    npm install pm2 -g
~~~
2. 启动服务
~~~shell
    pm2 start --env
    pm2 start --env test
    pm2 start --env dev
~~~
3. 查看运行状态
~~~shell
    pm2 list
~~~
4. 查看运行日志
~~~shell
    pm2 logs
~~~
